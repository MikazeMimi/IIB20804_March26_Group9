# IIB20804_March26_Group9

This is an IoT Application Security mini group project with contributions of 4 Unikl student; Syahmi.S, S.M.Aizat, Amir.F & W.A.Aiman.

Project Title: Intelligent Privacy-Preserving Indoor Surveillance Network.

Project Brief:-
A secure smart home application designed to provide indoor surveillance while strictly adhering to IoT security standards like TLS/SSL for data transmission verification and secure MQTTS communication between layers; Perception - Networking - Middleware - Application. Using an ESP32 and PIR sensors, the system detects motion and manages device states through a oneM2M CSE platform to ensure that video recording only occurs during authorized security modes, thereby preserving occupant privacy. The project involves a full development lifecycle—from threat modeling and DFD design to implementing device management via oneM2M APIs—to create a functional, interoperable, and vulnerability-tested security solution.
